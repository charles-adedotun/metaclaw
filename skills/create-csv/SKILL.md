# Create CSV and Markdown Files

You have Node.js `fs` and Python `pandas` available.

## When to Use Which Format

| Format | Best For |
|--------|----------|
| CSV | Data exports, spreadsheet-ready tables, API-style data dumps |
| Markdown | Readable reports, summaries, meeting notes, anything text-heavy |

## CSV — Best Practices

```javascript
const fs = require('fs');

// Always quote fields that might contain commas, newlines, or quotes
function csvRow(fields) {
  return fields.map(f => {
    const s = String(f ?? '');
    return s.includes(',') || s.includes('"') || s.includes('\n')
      ? `"${s.replace(/"/g, '""')}"` : s;
  }).join(',');
}

const headers = ["Name", "Value", "Status", "Date"];
const rows = [
  ["Server Alpha", "99.9%", "Active", "2026-02-25"],
  ["Server Beta", "97.2%", "Warning", "2026-02-25"],
];

const csv = [csvRow(headers), ...rows.map(csvRow)].join('\n');
fs.writeFileSync('/workspace/group/export.csv', csv);
```

Or with Python pandas:
```python
import pandas as pd
df = pd.DataFrame(data, columns=headers)
df.to_csv('/workspace/group/export.csv', index=False)
```

## Markdown — Best Practices

```javascript
const fs = require('fs');
const today = new Date().toISOString().split('T')[0];

const md = `# Report Title

*Prepared by ${process.env.ASSISTANT_NAME || "Assistant"} • ${today}*

---

## Section 1

Key findings paragraph here. Keep it scannable — short paragraphs, clear language.

- **Point one:** Brief explanation
- **Point two:** Brief explanation
- **Point three:** Brief explanation

## Section 2

| Metric | Value | Trend |
|--------|-------|-------|
| Uptime | 99.9% | ↑ |
| Response | 45ms | → |
| Errors | 0.1% | ↓ |

## Summary

Concluding paragraph with key takeaways and next steps.

---
*Generated by ${process.env.ASSISTANT_NAME || "Assistant"} — Chief of Staff*
`;

fs.writeFileSync('/workspace/group/report.md', md);
```

## Quality Checklist
- [ ] CSV: Headers present, fields properly quoted, no trailing commas
- [ ] Markdown: Title with date, horizontal rules as dividers, tables aligned
- [ ] File saved to `/workspace/group/` before send_file
